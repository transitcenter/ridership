# ridership

Visualization of US public transportation ridership data along with key population and socioeconomic variables.

The primary files needed when creating the production app are:

- `/app/index.html`
- `/app/assets/css`
- `/app/assets/images`
- `/app/assets/js`

## Getting Started

*Install dependencies*
run `yarn install`

*Dev build:*
run `grunt`

*Dist build:*
run `grunt build`
